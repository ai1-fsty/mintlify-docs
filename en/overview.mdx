Starrise AI is a unified AI API platform that gives developers access to the world's best models — including chat, image generation, video generation, and audio — through a single, consistent interface.

Instead of managing separate API keys, SDKs, and authentication flows for each provider, Starrise AI abstracts everything into one endpoint. Switch between models from OpenAI, Anthropic, ByteDance, Google, DeepSeek, and more by changing a single parameter.

## What Starrise AI Provides

**Unified access to 100+ models**
Chat completions, image generation, video generation, audio — all available under one API key. No need to integrate each provider separately.

**OpenAI-compatible interface**
For chat models, Starrise AI follows the OpenAI API specification. If you're already using the OpenAI SDK, switching to Starrise AI requires only a base URL change.

**Multi-modal support**
Beyond chat, Starrise AI supports image and video generation models with provider-specific parameters exposed in a clean, documented format.

**Predictable error handling**
Every endpoint returns consistent error codes and message formats, regardless of the underlying provider — so your error handling logic stays simple.

**Transparent pricing**
No platform fees or markup surprises. You pay per token for chat models and per generation for media models, billed based on actual usage.

## Supported Capabilities

| Category | Endpoints | Description |
| --- | --- | --- |
| **Chat** | `/v1/chat/completions` | Multi-turn, streaming, function calling, vision |
| **Responses** | `/v1/responses` | OpenAI Responses API |
| **Claude** | `/v1/messages` | Anthropic Messages API with extended thinking |
| **Gemini** | `/v1beta/models/*` | Google Gemini native API with search grounding |
| **Embeddings** | `/v1/embeddings`, `/v1/rerank` | Text embeddings and document reranking |
| **Images** | `/v1/images/*` | Text-to-image generation and editing |
| **Video** | `/v1/video/*`, `/mj/*` | Text-to-video, image-to-video, task management |
| **Audio** | `/v1/audio/*` | Speech-to-text, text-to-speech, translation |
| **Realtime** | `/v1/realtime` | WebSocket voice and real-time chat |
| **Midjourney** | `/mj/*` | Image generation, variations, describe |
| **Suno** | `/suno/*` | AI music generation |

## Three Steps to Integrate

1. **Create an account** — sign up in the console
2. **Create an API token** — create one under **Tokens** and copy your `sk-...` key
3. **Set Base URL** — copy the API address from the console home page into your SDK `base_url`

## Base URL

Copy your **API address** from the console home page:

```
{{api-base-url}}
```

<Info>

If your app already uses the OpenAI SDK, change `base_url` and `api_key` only. No code refactor required.

</Info>
