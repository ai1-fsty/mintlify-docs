Starrise AI supports video generation from multiple providers, including ByteDance Seedance, Kuaishou Kling, Alibaba Wan/HappyHorse, and others. Video generation is asynchronous: submit a task, receive a `task_id`, then poll for results.

## Endpoints

| Endpoint | Method | Description |
| --- | --- | --- |
| `/v1/video/generations` | POST | Create a video generation task |
| `/v1/video/generations/{task_id}` | GET | Query task status and retrieve results |

## Create Video Task

```bash
curl {{api-base-url}}/video/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "seedance-2.0",
    "prompt": "A cat walking through a futuristic city at night, neon lights reflecting on wet streets",
    "duration": 5,
    "aspect_ratio": "16:9"
  }'
```

### Text-to-Video Parameters

| Parameter | Type | Required | Description |
| --- | --- | --- |
| `model` | string | Yes | Model ID (see available models below) |
| `prompt` | string | Yes | Text description of the desired video |
| `duration` | integer | No | Video duration in seconds |
| `aspect_ratio` | string | No | Aspect ratio (e.g. `16:9`, `9:16`, `1:1`) |
| `resolution` | string | No | Output resolution (e.g. `720p`, `1080p`) |
| `negative_prompt` | string | No | Things to avoid in the video |
| `seed` | integer | No | Random seed for reproducibility |

### Image-to-Video Parameters

```bash
curl {{api-base-url}}/video/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "kling-v3-i2v",
    "image": "https://example.com/first-frame.jpg",
    "prompt": "The person starts walking forward",
    "duration": 5
  }'
```

Additional parameters for image-to-video:

| Parameter | Type | Required | Description |
| --- | --- | --- |
| `image` | string | Yes | URL or base64 of the reference image |
| `last_frame` | string | No | URL of the last frame image for frame control |
| `images` | array | No | Multiple reference images (model-dependent) |

## Query Task Status

```bash
curl {{api-base-url}}/video/generations/{task_id} \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Response (Processing)

```json
{
  "id": "vid-abc123",
  "status": "processing",
  "created_at": 1710000000
}
```

### Response (Completed)

```json
{
  "id": "vid-abc123",
  "status": "completed",
  "created_at": 1710000000,
  "completed_at": 1710000120,
  "video": {
    "url": "https://...",
    "duration": 5,
    "resolution": "1080p"
  }
}
```

### Task Statuses

| Status | Description |
| --- | --- |
| `pending` | Task queued, waiting to start |
| `processing` | Video generation in progress |
| `completed` | Generation finished, video URL available |
| `failed` | Generation failed, check error message |

| --- | --- | --- |
| `seedance-2.0` | ByteDance | Text-to-video & image-to-video |

| `kling-v3-t2v` | Kuaishou | Text-to-video |
| `kling-v3-i2v` | Kuaishou | Image-to-video |
| `kling-v3-omni` | Kuaishou | Omni video generation |
| `wan2.2-t2v-a14b` | Alibaba | Text-to-video |
| `wan2.2-i2v-a14b` | Alibaba | Image-to-video |
| `happyhorse-1.0-t2v` | Alibaba | Text-to-video |
| `happyhorse-1.0-i2v` | Alibaba | Image-to-video |
| `sora-2` | OpenAI | Text-to-video & image-to-video |
