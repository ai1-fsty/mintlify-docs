**Endpoint:** `POST /v1/messages`

Access Anthropic Claude models through the native Messages API format, with support for extended thinking, vision, and tool use.

## Authentication

Two authentication methods are supported:

**Standard gateway token:**
```bash
-H "Authorization: Bearer YOUR_API_KEY"
```

**Native Anthropic headers (auto-mapped by the gateway):**
```bash
-H "x-api-key: YOUR_API_KEY" \
-H "anthropic-version: 2023-06-01"
```

## Basic Request

```bash
curl {{api-base-url}}/messages \
  -H "x-api-key: YOUR_API_KEY" \
  -H "anthropic-version: 2023-06-01" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "claude-sonnet-4-6",
    "max_tokens": 1024,
    "messages": [
      {"role": "user", "content": "Explain the difference between REST and GraphQL."}
    ]
  }'
```

## Extended Thinking

Enable extended thinking for complex reasoning tasks:

```json
{
  "model": "claude-opus-4-8",
  "max_tokens": 4096,
  "thinking": {
    "type": "enabled",
    "budget_tokens": 2000
  },
  "messages": [
    {"role": "user", "content": "Solve this complex math problem..."}
  ]
}
```

## Vision

```json
{
  "model": "claude-sonnet-4-6",
  "max_tokens": 1024,
  "messages": [{
    "role": "user",
    "content": [
      {"type": "text", "text": "Describe this image in detail."},
      {"type": "image", "source": {
        "type": "base64",
        "media_type": "image/jpeg",
        "data": "/9j/..."
      }}
    ]
  }]
}
```

## SDK Usage

Use the official Anthropic SDK with Starrise AI:

```python
import anthropic

client = anthropic.Anthropic(
    base_url="{{api-origin}}",
    api_key="YOUR_API_KEY",
)

message = client.messages.create(
    model="claude-sonnet-4-6",
    max_tokens=1024,
    messages=[{"role": "user", "content": "Hello, Claude!"}]
)
print(message.content)
```
