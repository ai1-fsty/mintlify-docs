## List Models (OpenAI Format)

```bash
curl {{api-base-url}}/models \
  -H "Authorization: Bearer YOUR_API_KEY"
```

Response follows OpenAI `list` object shape with `data[]` entries:

```json
{
  "object": "list",
  "data": [
    {
      "id": "gpt-4o",
      "object": "model",
      "created": 1710000000,
      "owned_by": "openai"
    }
  ]
}
```

## Retrieve Model

```bash
curl {{api-base-url}}/models/gpt-4o \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Claude Model List

Use Anthropic-style headers:

```bash
curl {{api-base-url}}/models \
  -H "x-api-key: YOUR_API_KEY" \
  -H "anthropic-version: 2023-06-01"
```

## Gemini Model List

```bash
curl "{{api-origin}}/v1beta/models?key=YOUR_API_KEY"
```

Or with header:

```bash
curl {{api-origin}}/v1beta/models \
  -H "x-goog-api-key: YOUR_API_KEY"
```

OpenAI-compatible Gemini list:

```bash
curl {{api-origin}}/v1beta/openai/models \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Console

Browse full pricing and availability under **Pricing** in the console. Model availability depends on your account and whether **model limits** are enabled on your token.

<Info>

The model list returned by the API reflects models available to your specific token. Models restricted by token settings or account tier will not appear.

</Info>
