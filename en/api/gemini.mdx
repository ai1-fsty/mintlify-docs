Access Google Gemini models through the native Gemini API format, with support for thinking mode, Google Search grounding, image generation, and streaming.

## List Models

```bash
curl "{{api-origin}}/v1beta/models?key=YOUR_API_KEY"
```

## Generate Content

Path format: `POST /v1beta/models/{model}:{action}`

```bash
curl "{{api-origin}}/v1beta/models/gemini-2.5-flash:generateContent?key=YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "contents": [{
      "parts": [{"text": "Explain quantum computing in one sentence."}]
    }]
  }'
```

## Authentication

Multiple authentication methods are supported:

- `Authorization: Bearer YOUR_API_KEY`
- Header `x-goog-api-key: YOUR_API_KEY`
- Query parameter `?key=YOUR_API_KEY`

## Thinking Mode

Enable thinking for complex reasoning with Gemini 2.5/3.x models:

```json
{
  "contents": [{
    "parts": [{"text": "Solve this complex problem step by step."}]
  }],
  "generationConfig": {
    "thinkingConfig": {
      "thinkingBudget": 1024
    }
  }
}
```

## Google Search Grounding

```json
{
  "contents": [{
    "parts": [{"text": "What are the latest developments in AI?"}]
  }],
  "tools": [{
    "googleSearch": {}
  }]
}
```

## Streaming

Streaming uses SSE format:

```python
import google.generativeai as genai

genai.configure(
    api_key="YOUR_API_KEY",
    client_options={"api_endpoint": "{{api-origin}}"}
)

model = genai.GenerativeModel("gemini-2.5-flash")
response = model.generate_content("Tell me a story", stream=True)
for chunk in response:
    print(chunk.text, end="")
```

## OpenAI-Compatible Path

Some clients use OpenAI-style paths:

```
POST /v1/models/{model}:generateContent
```

Also supported via the gateway relay.

## Available Endpoints

| Path Pattern | Method | Description |
| --- | --- | --- |
| `/v1beta/models` | GET | List models |
| `/v1beta/models/{model}:generateContent` | POST | Generate content |
| `/v1beta/models/{model}:streamGenerateContent` | POST | Stream content |
| `/v1beta/openai/models` | GET | OpenAI-compatible model list |
| `/v1/models/{model}:generateContent` | POST | OpenAI path alias |
