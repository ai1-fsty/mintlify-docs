## About ChatHub

ChatHub is a popular LLM aggregation plugin that lets you chat with multiple models at the same time, making it easy to compare responses. ChatHub has hundreds of thousands of active users worldwide.

## Installing ChatHub

Open the [ChatHub official website](https://chathub.gg/) in your browser and click the "Add to Chrome" button to install the ChatHub browser extension.

## Using Starrise AI Models in ChatHub

1. In ChatHub settings, find the "Custom Bots" module and click the "Add" button.
2. In the dialog, fill in the following:
   * Enter a bot name.
   * Select **OpenAI Compatible** as the provider.
   * Enter your Starrise AI API key.
   * Enter the Starrise AI API endpoint: `https://ai.alad.com/v1`.
   * Enter a Starrise AI supported model (e.g. `gpt-4o`, `claude-opus-4-7`).
3. Click "Confirm" to finish configuring the model.
4. Start chatting.

## Multi-Model Comparison in ChatHub

You can repeat the steps above to add more models to ChatHub, then use ChatHub's All-in-One feature to chat with multiple models simultaneously (up to 6 models at once).
