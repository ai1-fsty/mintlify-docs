As the 2023 "Chrome Editor's Choice Extension", Sider has received over 50,000 five-star reviews and has more than 6 million active users. The Sider browser extension primarily addresses:

* Sider (ChatGPT Sidebar) is your personal AI assistant that you can use while browsing any website.

<Note>
  - Developers can visit the [Sider official website](https://sider.ai/en/) to use the Sider application.
  - Sider provides an OpenAI API-compatible model registration method to meet the needs of software developers, making it more convenient to use the required models.
  - As a one-stop cloud service platform integrating top AI models, Starrise AI is committed to providing developers with faster, cheaper, more comprehensive, and smoother model APIs.
</Note>

## 1. Get Your API Key

* Open the [Starrise AI Console](https://ai.alad.com/) and register an account (or log in if you already have one).
* After registering, go to **API Keys**, create a new API key, and copy it for later use.

## 2. Using Starrise AI Language Models in Sider

* Open Sider settings.
* Use the **OpenAI API Key** mode.
* Enter the API Key obtained from Starrise AI.
* Enter the corresponding Starrise AI API endpoint: `https://ai.alad.com/v1`
* Enter the model name from Starrise AI (e.g. `gpt-4o`, `claude-opus-4-7`).
* Verify in the chat dialog to make sure everything is working correctly.
