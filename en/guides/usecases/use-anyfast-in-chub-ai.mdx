## About Chub AI

Chub AI is a popular AI tool that supports custom API providers. You can connect it to Starrise AI to access a wide range of AI models.

## Get Your API Key

Open the [Starrise AI Console](https://ai.alad.com/) and register an account (or log in if you already have one). Go to **API Keys**, create a new API key, and copy it.

## Configure Starrise AI in Chub AI

1. Open Chub AI settings.
2. Select **OpenAI Compatible** or **Custom API Provider** as the provider type.
3. Enter your Starrise AI API Key.
4. Set the **API Base URL** to `https://ai.alad.com/v1`.
5. Enter the model name (e.g. `gpt-4o`, `claude-opus-4-7`).
6. Save the configuration, and you're ready to use Starrise AI models!

## Available Models

Starrise AI offers a wide selection of models, including GPT-4o, Claude Opus 4.7, Gemini, and more. Visit the [Starrise AI Console](https://ai.alad.com/) to view all available models.
