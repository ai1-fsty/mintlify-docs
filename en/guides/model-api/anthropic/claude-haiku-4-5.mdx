Claude Haiku 4.5 is Anthropic's fastest and most cost-effective model, available through Starrise AI via OpenAI-compatible interface. It responds to everyday tasks with ultra-low latency, making it ideal for high-throughput and latency-sensitive applications.

## Key Capabilities

* **OpenAI-compatible** — Drop-in replacement for the OpenAI SDK, no other code changes needed
* **Ultra-fast response** — Optimized for low latency
* **Cost-effective** — Far lower per-token cost than larger models
* **Streaming** — Real-time token streaming via SSE
* **Capable** — Excellent performance on coding, writing, and analysis tasks

## Quick Example

<CodeGroup>
  ```bash cURL
  curl https://ai.alad.com/v1/messages \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{
      "model": "claude-haiku-4-5",
      "messages": [
        { "role": "user", "content": "Explain quantum entanglement in simple terms." }
      ]
    }'
  ```

  ```python Python
  from openai import OpenAI

  client = OpenAI(
      api_key="YOUR_API_KEY",
      base_url="https://ai.alad.com/v1"
  )

  response = client.chat.completions.create(
      model="claude-haiku-4-5",
      messages=[
          {"role": "user", "content": "Explain quantum entanglement in simple terms."}
      ]
  )

  print(response.choices[0].message.content)
  ```

  ```python Streaming
  from openai import OpenAI

  client = OpenAI(
      api_key="YOUR_API_KEY",
      base_url="https://ai.alad.com/v1"
  )

  stream = client.chat.completions.create(
      model="claude-haiku-4-5",
      messages=[
          {"role": "user", "content": "Write a short poem about the ocean."}
      ],
      stream=True
  )

  for chunk in stream:
      print(chunk.choices[0].delta.content or "", end="")
  ```
</CodeGroup>

## Parameters

| Parameter | Type | Required | Description |
| ------------- | -------------- | -- | ------------------------ |
| `model` | string | Yes | Fixed value: `claude-haiku-4-5` |
| `messages` | array | Yes | Array of `{ role, content }` objects |
| `max_tokens` | integer | No | Maximum number of tokens to generate |
| `temperature` | float | No | `0`–`2`, controls randomness, default `1` |
| `stream` | boolean | No | Enable SSE streaming, default `false` |
| `top_p` | float | No | Nucleus sampling threshold, default `1` |
| `stop` | string / array | No | Sequences where generation stops |
