## Configuration

| Setting | OpenAI Default | Starrise AI |
| --- | --- | --- |
| Base URL | `https://api.openai.com/v1` | `{{api-base-url}}` |
| API Key | OpenAI API key | Starrise AI token (`sk-...`) |

Sign in to the console and copy your **API address** (Base URL) from the home page. Paste it into your SDK or third-party tool as the Base URL / API Host:

```
{{api-base-url}}
```

<Info>

When using the official OpenAI SDK, make sure the Base URL ends with `/v1`.

</Info>

## Official SDKs

### Python

```bash
pip install openai
```

```python
from openai import OpenAI

client = OpenAI(
    base_url="{{api-base-url}}",
    api_key="sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
)
```

### Node.js

```bash
npm install openai
```

```javascript
import OpenAI from "openai";

const client = new OpenAI({
  baseURL: "{{api-base-url}}",
  apiKey: "sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
});
```

## OpenAI-Compatible Endpoints

These routes use `Authorization: Bearer sk-...`.

| Endpoint | Method | Description |
| --- | --- | --- |
| `/v1/chat/completions` | POST | Chat completions (streaming, tools, vision) |
| `/v1/completions` | POST | Legacy text completions |
| `/v1/responses` | POST | OpenAI Responses API |
| `/v1/responses/compact` | POST | Responses compaction |
| `/v1/embeddings` | POST | Text embeddings |
| `/v1/rerank` | POST | Document reranking (Jina-compatible) |
| `/v1/images/generations` | POST | Image generation |
| `/v1/images/edits` | POST | Image editing |
| `/v1/edits` | POST | Legacy image edit |
| `/v1/audio/transcriptions` | POST | Speech-to-text |
| `/v1/audio/translations` | POST | Audio translation |
| `/v1/audio/speech` | POST | Text-to-speech |
| `/v1/moderations` | POST | Content moderation |
| `/v1/models` | GET | List models (OpenAI format) |
| `/v1/models/:model` | GET | Retrieve model details |
| `/v1/realtime` | GET (WebSocket) | OpenAI Realtime API |

<Info title="Not implemented">

`/v1/images/variations`, `/v1/files/*`, and `/v1/fine-tunes/*` return not-implemented on this gateway.

</Info>

## Video Generation Endpoints

| Endpoint | Method | Description |
| --- | --- | --- |
| `/v1/video/generations` | POST | Create video generation task |
| `/v1/video/generations/{task_id}` | GET | Query video task status |

Video generation is asynchronous. Submit a task to receive a `task_id`, then poll the query endpoint for results.

## Claude (Anthropic) Endpoints

| Endpoint | Method | Auth |

| `/v1/messages` | POST | `Authorization: Bearer sk-...` or `x-api-key` + `anthropic-version` |

## Troubleshooting

| Issue | Solution |
| --- | --- |
| `401 Unauthorized` | Check token; use `Bearer sk-...` |
| `403 Forbidden` | Model/IP restriction or account issue |
| `404 Model not found` | Call `GET /v1/models` or check console pricing |
| Connection timeout | Ensure base URL ends with `/v1` for OpenAI SDK |
| SDK still hits OpenAI | Set `base_url` / `baseURL` explicitly |
| `429 Rate limited` | Implement exponential backoff; reduce request rate |
