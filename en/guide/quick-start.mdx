Starrise AI provides a unified API to access AI models from multiple providers — including chat, image generation, video generation, and audio. This guide gets you up and running with your first API call.

## Prerequisites

- A Starrise AI account ([sign up in the console]({{api-origin}}))
- An API token ([Authentication](/en/guide/authentication))

## Step 1: Create an Account

1. Open the Starrise AI console at [{{api-origin}}]({{api-origin}})
2. Register with email or supported OAuth (GitHub, Discord, etc.)
3. New accounts may receive starter quota (as shown in your account dashboard)

## Step 2: Create an API Token

1. Open **Tokens** in the console
2. Click **Create Token**
3. Set a name and optional quota, expiration, and other limits
4. Submit and **Copy** the key from the list — format: `sk-...`

<Warning>

The full key is shown once. Store it securely.

</Warning>

## Step 3: Make Your First Request

Use your API key to call a chat model. Replace `YOUR_API_KEY` with your actual token:

```bash
curl {{api-base-url}}/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "gpt-4o",
    "messages": [
      { "role": "user", "content": "Hello!" }
    ]
  }'
```

Example response:

```json
{
  "id": "chatcmpl-xxx",
  "object": "chat.completion",
  "model": "gpt-4o",
  "choices": [{
    "index": 0,
    "message": {
      "role": "assistant",
      "content": "Hello! How can I help you today?"
    },
    "finish_reason": "stop"
  }],
  "usage": {
    "prompt_tokens": 10,
    "completion_tokens": 12,
    "total_tokens": 22
  }
}
```

## Use the SDK

### Python

```bash
pip install openai
```

```python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="{{api-base-url}}"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello!"}]
)

print(response.choices[0].message.content)
```

### Node.js

```bash
npm install openai
```

```javascript
import OpenAI from "openai";

const client = new OpenAI({
  apiKey: "YOUR_API_KEY",
  baseURL: "{{api-base-url}}",
});

const response = await client.chat.completions.create({
  model: "gpt-4o",
  messages: [{ role: "user", content: "Hello!" }],
});

console.log(response.choices[0].message.content);
```

## Streaming

Set `"stream": true` to receive Server-Sent Events (SSE) with `data: {...}` chunks:

```python
stream = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Tell me a story"}],
    stream=True
)

for chunk in stream:
    if chunk.choices[0].delta.content:
        print(chunk.choices[0].delta.content, end="")
```

## Error Handling

| Status | Description |
| --- | --- |
| `200` | Success |
| `400` | Bad request — invalid parameters |
| `401` | Unauthorized — invalid or missing API key |
| `403` | Forbidden — insufficient permissions or model restriction |
| `404` | Not found — invalid endpoint or model |
| `429` | Rate limit exceeded — too many requests |
| `500` | Internal server error |

When an error occurs, the API returns a JSON response with an `error` object:

```json
{
  "error": {
    "message": "Invalid API key provided.",
    "type": "authentication_error",
    "code": "invalid_api_key"
  }
}
```
