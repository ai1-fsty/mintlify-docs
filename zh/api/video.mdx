title: "视频生成"

Starrise AI 支持来自多个供应商的视频生成，包括 ByteDance Seedance、快手 Kling、阿里巴巴 Wan/HappyHorse 等。视频生成为异步模式：提交任务后获取 `task_id`，然后轮询查询结果。

## 端点

| 端点 | 方法 | 说明 |
| --- | --- | --- |
| `/v1/video/generations` | POST | 创建视频生成任务 |
| `/v1/video/generations/{task_id}` | GET | 查询任务状态及获取结果 |

## 创建视频任务

```bash
curl {{api-base-url}}/video/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "seedance-2.0",
    "prompt": "一只猫走过夜晚的未来城市，霓虹灯反射在湿润的街道上",
    "duration": 5,
    "aspect_ratio": "16:9"
  }'
```

### 文生视频参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `model` | string | 是 | 模型 ID（见下方可用模型列表） |
| `prompt` | string | 是 | 期望视频的文字描述 |
| `duration` | integer | 否 | 视频时长（秒） |
| `aspect_ratio` | string | 否 | 画面比例（如 `16:9`、`9:16`、`1:1`） |
| `resolution` | string | 否 | 输出分辨率（如 `720p`、`1080p`） |
| `negative_prompt` | string | 否 | 视频中避免出现的内容 |
| `seed` | integer | 否 | 随机种子，用于结果复现 |

### 图生视频参数

```bash
curl {{api-base-url}}/video/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "kling-v3-i2v",
    "image": "https://example.com/first-frame.jpg",
    "prompt": "画面中的人开始向前走",
    "duration": 5
  }'
```

图生视频额外参数：

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `image` | string | 是 | 参考图片 URL 或 base64 |
| `last_frame` | string | 否 | 尾帧图片 URL，用于首尾帧控制 |
| `images` | array | 否 | 多张参考图片（取决于模型） |

## 查询任务状态

```bash
curl {{api-base-url}}/video/generations/{task_id} \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### 处理中响应

```json
{
  "id": "vid-abc123",
  "status": "processing",
  "created_at": 1710000000
}
```

### 完成响应

```json
{
  "id": "vid-abc123",
  "status": "completed",
  "created_at": 1710000000,
  "completed_at": 1710000120,
  "video": {
    "url": "https://...",
    "duration": 5,
    "resolution": "1080p"
  }
}
```

### 任务状态

| 状态 | 说明 |
| --- | --- |
| `pending` | 任务已排队，等待开始 |
| `processing` | 视频生成进行中 |
| `completed` | 生成完成，视频 URL 可用 |
| `failed` | 生成失败，请查看错误信息 |

## 可用视频模型

| 模型 | 供应商 | 类型 |
| --- | --- | --- |
| `seedance-2.0` | ByteDance | 文生视频与图生视频 |
| `seedance-2.0-ultra` | ByteDance | 高质量视频生成 |
| `seedance-2.0-fast` | ByteDance | 快速视频生成 |
| `kling-v3-t2v` | 快手 | 文生视频 |
| `kling-v3-i2v` | 快手 | 图生视频 |
| `kling-v3-omni` | 快手 | Omni 视频生成 |
| `wan2.2-t2v-a14b` | 阿里巴巴 | 文生视频 |
| `wan2.2-i2v-a14b` | 阿里巴巴 | 图生视频 |
| `happyhorse-1.0-t2v` | 阿里巴巴 | 文生视频 |
| `happyhorse-1.0-i2v` | 阿里巴巴 | 图生视频 |
| `sora-2` | OpenAI | 文生视频与图生视频 |

完整列表、定价与可用性请查看控制台 **模型定价** 页面。
