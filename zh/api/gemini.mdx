通过原生 Gemini API 格式访问 Google Gemini 模型，支持思考模式、Google 搜索增强、图片生成和流式输出。

## 模型列表

```bash
curl "{{api-origin}}/v1beta/models?key=YOUR_API_KEY"
```

## 生成内容

路径格式：`POST /v1beta/models/{model}:{action}`

```bash
curl "{{api-origin}}/v1beta/models/gemini-2.5-flash:generateContent?key=YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "contents": [{
      "parts": [{"text": "用一句话解释量子计算。"}]
    }]
  }'
```

## 认证方式

支持多种认证方式：

- `Authorization: Bearer YOUR_API_KEY`
- 请求头 `x-goog-api-key: YOUR_API_KEY`
- 查询参数 `?key=YOUR_API_KEY`

## 思考模式

为 Gemini 2.5/3.x 模型启用深度思考：

```json
{
  "contents": [{
    "parts": [{"text": "逐步解决这个复杂问题。"}]
  }],
  "generationConfig": {
    "thinkingConfig": {
      "thinkingBudget": 1024
    }
  }
}
```

## Google 搜索增强

```json
{
  "contents": [{
    "parts": [{"text": "人工智能领域有哪些最新进展？"}]
  }],
  "tools": [{
    "googleSearch": {}
  }]
}
```

## 流式输出

流式输出使用 SSE 格式：

```python
import google.generativeai as genai

genai.configure(
    api_key="YOUR_API_KEY",
    client_options={"api_endpoint": "{{api-origin}}"}
)

model = genai.GenerativeModel("gemini-2.5-flash")
response = model.generate_content("讲一个故事", stream=True)
for chunk in response:
    print(chunk.text, end="")
```

## OpenAI 兼容路径

部分客户端可使用 OpenAI 风格路径：

```
POST /v1/models/{model}:generateContent
```

网关同样支持此路径。

## 可用端点

| 路径模式 | 方法 | 说明 |
| --- | --- | --- |
| `/v1beta/models` | GET | 模型列表 |
| `/v1beta/models/{model}:generateContent` | POST | 生成内容 |
| `/v1beta/models/{model}:streamGenerateContent` | POST | 流式生成 |
| `/v1beta/openai/models` | GET | OpenAI 兼容模型列表 |
| `/v1/models/{model}:generateContent` | POST | OpenAI 路径别名 |
