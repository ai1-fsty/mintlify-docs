**端点:** `GET /v1/realtime` (WebSocket 升级)

Realtime API 通过 WebSocket 实现低延迟、多模态交互，支持实时语音对话、文字聊天和函数调用。

## 认证方式

网关从 WebSocket 子协议头读取 API 密钥：

```
Sec-WebSocket-Protocol: realtime, openai-insecure-api-key.YOUR_API_KEY, openai-beta.realtime-v1
```

若客户端支持，也可在升级前设置 `Authorization: Bearer YOUR_API_KEY`。

## 连接地址

```
{{ws-base-url}}/v1/realtime?model=gpt-4o-realtime-preview
```

## OpenAI SDK 用法

```python
from openai import OpenAI

client = OpenAI(
    base_url="{{api-base-url}}",
    api_key="YOUR_API_KEY",
)

# 连接 Realtime API
# SDK 自动管理 WebSocket 连接和事件处理
```

## 支持的功能

| 功能 | 说明 |
| --- | --- |
| **语音对话** | 实时语音到语音交互 |
| **文字聊天** | 发送和接收文字消息 |
| **函数调用** | 模型可实时调用您的函数 |
| **会话配置** | 设置指令、语音、模态和工具 |
| **音频输入/输出** | PCM16 音频，可配置采样率 |
| **中断** | 取消进行中的模型响应 |

## 会话配置

```json
{
  "type": "session.update",
  "session": {
    "modalities": ["text", "audio"],
    "instructions": "你是一个有用的助手。",
    "voice": "alloy",
    "input_audio_format": "pcm16",
    "output_audio_format": "pcm16",
    "turn_detection": {
      "type": "server_vad",
      "threshold": 0.5,
      "prefix_padding_ms": 300,
      "silence_duration_ms": 500
    }
  }
}
```
