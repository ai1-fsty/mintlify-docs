## 列表（OpenAI 格式）

```bash
curl {{api-base-url}}/models \
  -H "Authorization: Bearer YOUR_API_KEY"
```

响应遵循 OpenAI `list` 结构，`data[]` 含模型信息：

```json
{
  "object": "list",
  "data": [
    {
      "id": "gpt-4o",
      "object": "model",
      "created": 1710000000,
      "owned_by": "openai"
    }
  ]
}
```

## 获取单个模型

```bash
curl {{api-base-url}}/models/gpt-4o \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Claude 模型列表

使用 Anthropic 风格请求头：

```bash
curl {{api-base-url}}/models \
  -H "x-api-key: YOUR_API_KEY" \
  -H "anthropic-version: 2023-06-01"
```

## Gemini 模型列表

```bash
curl "{{api-origin}}/v1beta/models?key=YOUR_API_KEY"
```

或使用请求头：

```bash
curl {{api-origin}}/v1beta/models \
  -H "x-goog-api-key: YOUR_API_KEY"
```

OpenAI 兼容列表：

```bash
curl {{api-origin}}/v1beta/openai/models \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## 控制台

在控制台 **模型定价** 页面查看完整定价与可用模型。模型可用性取决于您的账户权限及令牌是否启用了 **模型限制**。

<Info>

API 返回的模型列表反映了当前令牌可用的模型。被令牌设置或账户等级限制的模型不会出现在列表中。

</Info>
