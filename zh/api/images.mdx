| 端点 | 方法 | 说明 |
| --- | --- | --- |
| `/v1/images/generations` | POST | 文生图 |
| `/v1/images/edits` | POST | 带遮罩的图片编辑 |

## 生成图片

```bash
curl {{api-base-url}}/images/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "dall-e-3",
    "prompt": "夕阳下的宁静山湖，油画风格",
    "n": 1,
    "size": "1024x1024",
    "quality": "standard"
  }'
```

### 响应

```json
{
  "created": 1710000000,
  "data": [
    {
      "url": "https://...",
      "revised_prompt": "夕阳下的宁静山湖..."
    }
  ]
}
```

### Python SDK

```python
from openai import OpenAI

client = OpenAI(
    base_url="{{api-base-url}}",
    api_key="YOUR_API_KEY",
)

result = client.images.generate(
    model="dall-e-3",
    prompt="夕阳下的宁静山湖",
    size="1024x1024",
    quality="standard",
    n=1,
)
print(result.data[0].url)
```

## 参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `model` | string | 是 | 模型 ID（如 `dall-e-3`、`gpt-image-2`） |
| `prompt` | string | 是 | 期望图片的文字描述 |
| `n` | integer | 否 | 生成数量（1–10），默认 `1` |
| `size` | string | 否 | 图片尺寸（如 `1024x1024`、`1792x1024`） |
| `quality` | string | 否 | `standard` 或 `hd` |
| `style` | string | 否 | `vivid` 或 `natural` |
| `response_format` | string | 否 | `url` 或 `b64_json` |

## 编辑图片

使用 `POST /v1/images/edits` 配合 multipart 表单数据：

```bash
curl {{api-base-url}}/images/edits \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F image=@original.png \
  -F mask=@mask.png \
  -F prompt="在天空中加一道彩虹" \
  -F n=1 \
  -F size="1024x1024"
```
