---
title: "Responses API"
description: "面向 Agent 工作流的 OpenAI Responses API，内置工具与状态处理。"
---

| 端点 | 方法 |
| --- | --- |
| `/v1/responses` | POST |
| `/v1/responses/compact` | POST |

## 创建 Response

```bash
curl {{api-base-url}}/responses \
  -H "Authorization: Bearer sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "input": "总结最新项目进展。"
  }'
```

## 压缩

```bash
curl {{api-base-url}}/responses/compact \
  -H "Authorization: Bearer sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{ ... }'
```

在 OpenAI SDK 中将 `base_url` 指向 Starrise-api，使用 `client.responses.create(...)` 即可。

<Info>

是否可用取决于所选模型在上游通道是否支持 Responses API。

</Info>
