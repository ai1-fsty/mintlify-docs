| 端点 | 方法 | 说明 |
| --- | --- | --- |
| `/v1/responses` | POST | 创建带内置工具的响应 |
| `/v1/responses/compact` | POST | 压缩对话上下文 |

## 创建响应

```bash
curl {{api-base-url}}/responses \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "input": "总结最新的项目状态并列出待办事项。"
  }'
```

Responses API 提供更高级别的接口，自动处理工具执行、状态管理和对话上下文。非常适合智能代理工作流。

### Python SDK

```python
from openai import OpenAI

client = OpenAI(
    base_url="{{api-base-url}}",
    api_key="YOUR_API_KEY",
)

response = client.responses.create(
    model="gpt-4o",
    input="总结最新的项目状态。",
    tools=[{
        "type": "file_search",
        "vector_store_ids": ["vs_abc123"]
    }]
)
```

## 压缩

使用 `/v1/responses/compact` 压缩对话上下文，保留关键信息：

```bash
curl {{api-base-url}}/responses/compact \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{ ... }'
```

<Info>

可用性取决于上游渠道是否支持所选模型的 Responses 格式。

</Info>
