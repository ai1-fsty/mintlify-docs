Starrise AI 是一个统一的 AI API 平台，让开发者通过单一、一致的接口访问全球顶尖模型——涵盖对话、图片生成和视频生成。

无需为每家提供商单独管理 API Key、SDK 和认证流程，Starrise AI 将一切抽象为统一端点。只需修改一个参数，即可在 Anthropic、ByteDance、OpenAI、Google 等提供商的模型之间自由切换。

## Starrise AI 提供什么

**统一访问 100+ 模型**
对话补全、图片生成、视频生成——全部通过一个 API Key 访问，无需逐一集成各家提供商。

**OpenAI 兼容接口**
对话模型完全遵循 OpenAI API 规范。如果你已在使用 OpenAI SDK，切换到 Starrise AI 只需修改 Base URL，其余代码无需改动。

**多模态支持**
除对话外，Starrise AI 还支持图片和视频生成模型，各提供商的专属参数均以清晰、有文档的方式暴露。

**一致的错误处理**
无论底层调用哪家提供商，每个接口都返回统一的错误码和消息格式，让你的异常处理逻辑保持简洁。

**透明定价**
无平台费用，无隐藏加价。对话模型按 Token 计费，媒体模型按次计费，仅收取实际使用量。
