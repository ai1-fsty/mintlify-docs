## 支持的能力

| 类别 | 端点 | 说明 |
| --- | --- | --- |
| **对话** | `/v1/chat/completions` | 多轮、流式、函数调用 |
| **Responses** | `/v1/responses` | OpenAI Responses API |
| **Claude** | `/v1/messages` | Anthropic Messages API |
| **Gemini** | `/v1beta/models/*` | Google Gemini 原生 API |
| **向量** | `/v1/embeddings`、`/v1/rerank` | Embedding 与重排序 |
| **图片** | `/v1/images/*` | 文生图与编辑 |
| **音频** | `/v1/audio/*` | 转写、合成、翻译 |
| **Realtime** | `/v1/realtime` | WebSocket 实时对话 |
| **Midjourney** | `/mj/*` | 绘图与视频任务 |
| **Suno** | `/suno/*` | AI 音乐生成 |

## 三步接入

1. **创建账户** — 在控制台注册
2. **创建 API 密钥** — 在 **令牌** 页面创建并复制 `sk-...`
3. **配置 Base URL** — 从控制台首页复制 API 地址，填入 OpenAI SDK 的 `base_url`

## Base URL

登录控制台，在首页复制 **API 地址**。示例：

```
{{api-base-url}}
```

<Info>

若应用已使用 OpenAI SDK，通常只需修改 `base_url` 和 `api_key`，无需改造业务代码。

</Info>
