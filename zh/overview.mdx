Starrise AI 是统一的 AI API 平台，为开发者提供访问全球顶尖模型的能力 — 包括对话、图片生成、视频生成和音频 — 通过单一、一致的接口。

无需为每个供应商管理独立的 API 密钥、SDK 和认证流程，Starrise AI 将所有能力抽象为一个端点。只需修改一个参数，即可在 OpenAI、Anthropic、ByteDance、Google、DeepSeek 等模型之间自由切换。

## Starrise AI 提供的能力

**100+ 模型统一接入**
对话补全、图片生成、视频生成、音频 — 全部通过一个 API 密钥访问。无需分别对接各个供应商。

**OpenAI 兼容接口**
对话模型完全遵循 OpenAI API 规范。如果已在用 OpenAI SDK，切换到 Starrise AI 只需修改 Base URL。

**多模态支持**
除对话外，Starrise AI 还支持图片和视频生成模型，供应商特定参数以清晰、文档化的方式呈现。

**可预测的错误处理**
无论底层供应商是谁，每个端点都返回一致的错误码和消息格式 — 让您的错误处理逻辑保持简洁。

**透明定价**
无平台费用或加价。对话模型按 Token 计费，媒体模型按生成次数计费，基于实际用量。

## 支持的能力

| 类别 | 端点 | 说明 |
| --- | --- | --- |
| **对话** | `/v1/chat/completions` | 多轮、流式、函数调用、视觉 |
| **Responses** | `/v1/responses` | OpenAI Responses API |
| **Claude** | `/v1/messages` | Anthropic Messages API 与扩展思考 |
| **Gemini** | `/v1beta/models/*` | Google Gemini 原生 API 与搜索增强 |
| **向量** | `/v1/embeddings`、`/v1/rerank` | 文本向量与文档重排序 |
| **图片** | `/v1/images/*` | 文生图与编辑 |
| **视频** | `/v1/video/*`、`/mj/*` | 文生视频、图生视频、任务管理 |
| **音频** | `/v1/audio/*` | 语音转文字、文字转语音、翻译 |
| **Realtime** | `/v1/realtime` | WebSocket 实时语音对话 |
| **Midjourney** | `/mj/*` | 绘图、变体、图片描述 |
| **Suno** | `/suno/*` | AI 音乐生成 |

## 三步接入

1. **创建账户** — 在控制台注册
2. **创建 API 密钥** — 在 **令牌** 页面创建并复制 `sk-...`
3. **配置 Base URL** — 从控制台首页复制 API 地址，填入 SDK 的 `base_url`

## Base URL

登录控制台，在首页复制 **API 地址**：

```
{{api-base-url}}
```

<Info>

若应用已使用 OpenAI SDK，通常只需修改 `base_url` 和 `api_key`，无需改造业务代码。

</Info>
