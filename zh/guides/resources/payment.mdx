Starrise AI 采用预付费积分制。你向账户充值后，每次 API 调用会实时扣除积分——对话模型按 Token 计费，图片和视频生成模型按次计费。

## 账户充值

前往控制台的[充值页面](https://ai.starrise.tech/console/topup)为账户添加积分。

<Card title="立即充值" icon="circle-plus" href="https://ai.starrise.tech/console/topup" horizontal>
  为你的 Starrise AI 账户添加积分。
</Card>

## 支持的支付方式

* **信用卡 / 借记卡** — Visa、Mastercard、American Express
* **支付宝**
* **微信支付**

## 计费方式

* 积分在 API 调用时实时扣除
* 对话模型按 **Token 数**计费（输入 + 输出）
* 图片和视频生成模型按**生成次数**计费
* 未使用的积分永不过期

## 查看余额

你可以在[控制台](https://ai.starrise.tech/console/topup)查看当前积分余额和使用记录。
