## 关于 Hermes Agent

Hermes Agent 是一款流行的 AI 工具，支持自定义 API 提供商。你可以将其接入 Starrise AI 以使用丰富的 AI 模型。

## 获取 API Key

打开 [Starrise AI 控制台](https://ai.alad.com/) 并注册账号（如果注册过，直接登录即可）。前往 **API 密钥**，创建新的 API Key 并复制。

## 在 Hermes Agent 中配置 Starrise AI

1. 打开 Hermes Agent 设置。
2. 选择 **OpenAI Compatible** 或 **自定义 API 提供商** 作为提供商类型。
3. 输入 Starrise AI API Key。
4. 设置 **API Base URL** 为 `https://ai.alad.com/v1`。
5. 填入模型名称（如 `gpt-4o`、`claude-opus-4-7`）。
6. 保存配置，即可开始使用 Starrise AI 模型！

## 可用模型

Starrise AI 提供丰富的模型选择，包括 GPT-4o、Claude Opus 4.7、Gemini 等。访问 [Starrise AI 控制台](https://ai.alad.com/) 查看所有可用模型。
