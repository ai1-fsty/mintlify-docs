作为 2023 年"Chrome 编辑推荐扩展"，Sider 获得了超过 5 万条"五星好评"，并拥有超过 600 万活跃用户。Sider 浏览器扩展主要解决以下问题：

* Sider（ChatGPT 侧边栏）是您贴心的 AI 助手，可以在浏览任何网站时使用它。

<Note>
  - 开发者可以访问 [Sider 官方网站](https://sider.ai/en/) 使用 Sider 应用。
  - Sider 提供了一种兼容 OpenAI API 模型的注册方式，以满足软件开发者的需求，使使用所需模型更加便捷。
  - 作为集成顶级 AI 模型的一站式云服务平台，Starrise AI 致力于为开发者提供更快、更便宜、更全面、更流畅的模型 API。
</Note>

## 1. 获取 API Key

* 打开 [Starrise AI 控制台](https://ai.alad.com/) 并注册账号（如果已经注册，直接登录即可）。
* 注册完成后，前往 **API 密钥**，创建新的 API Key，并复制密钥以供后续使用。

## 2. 在 Sider 中使用 Starrise AI 语言模型

* 打开 Sider 设置。
* 使用 **OpenAI API Key** 模式。
* 输入从 Starrise AI 获取的 API Key。
* 输入对应的 Starrise AI API 端点地址：`https://ai.alad.com/v1`
* 输入来自 Starrise AI 的模型名称（如 `gpt-4o`、`claude-opus-4-7`）。
* 在对话框中进行验证以确保其正常工作。
