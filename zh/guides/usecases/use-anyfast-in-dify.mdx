## 1. 获取 API Key

打开 [Starrise AI 控制台](https://ai.alad.com/) 并注册账号（如果注册过，直接登录即可）。完成注册后，前往 **API 密钥**，创建新的 API Key，点击密钥进行复制，以备后续使用。

## 2. 在 Dify 中使用 Starrise AI 语言模型

### 2.1 添加 Starrise AI 为模型提供方

在 Dify 首页右上角选择"设置"，选择左上角"模型供应商"。

添加 **OpenAI 兼容接口** 的模型提供方。填写 Starrise AI 的接入配置：

* **Model Name**：从 Starrise AI 平台选择模型（如 `gpt-4o`、`claude-opus-4-7`）
* **API Key**：粘贴在 Starrise AI 控制台创建的 API Key
* **API 端点 URL**：`https://ai.alad.com/v1`

点击保存。校验成功后，可以在模型列表中看到 Starrise AI 提供的模型，并在应用中使用。

### 2.2 使用未在默认列表中的模型

* 打开 Dify 的"设置" → "模型供应商"
* 添加兼容 OpenAI 接口的模型服务平台
* 在其中填入对应的 Model Name、API Key、API 端点

## 3. 在 Dify 中使用 Starrise AI 生图模型

Starrise AI 的图像生成模型（如 `wan2.7-image-pro`）也可以通过 OpenAI 兼容接口在 Dify 中使用，配置方式与语言模型相同。
