Claude Opus 4.5 是 Anthropic 最智能、最强大的模型，通过 Starrise AI 以 OpenAI 兼容接口提供服务。在所有基准测试中表现顶尖，拥有最深度的推理、最强的编程能力和最细腻的创意写作。

## 核心能力

* **OpenAI 兼容** — 可直接替换 OpenAI SDK，无需修改其他代码
* **最强推理** — 在复杂多步骤问题上准确率最高
* **顶级编程** — 擅长大规模代码生成、调试和架构设计
* **卓越创造力** — 最细腻、最有表现力的写作和分析
* **长上下文** — 支持大文档处理和多轮对话
* **流式输出** — 通过 SSE 实现实时 Token 流式传输

## 快速示例

<CodeGroup>
  ```bash cURL
  curl https://ai.starrise.tech/v1/messages \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{
      "model": "claude-opus-4-5",
      "messages": [
        { "role": "user", "content": "用简单的语言解释量子纠缠。" }
      ]
    }'
  ```

  ```python Python
  from openai import OpenAI

  client = OpenAI(
      api_key="YOUR_API_KEY",
      base_url="https://ai.starrise.tech/v1"
  )

  response = client.chat.completions.create(
      model="claude-opus-4-5",
      messages=[
          {"role": "user", "content": "用简单的语言解释量子纠缠。"}
      ]
  )

  print(response.choices[0].message.content)
  ```

  ```python 流式输出
  from openai import OpenAI

  client = OpenAI(
      api_key="YOUR_API_KEY",
      base_url="https://ai.starrise.tech/v1"
  )

  stream = client.chat.completions.create(
      model="claude-opus-4-5",
      messages=[
          {"role": "user", "content": "写一首关于大海的短诗。"}
      ],
      stream=True
  )

  for chunk in stream:
      print(chunk.choices[0].delta.content or "", end="")
  ```
</CodeGroup>

## 参数说明

| 参数            | 类型             | 必填 | 说明                       |
| ------------- | -------------- | -- | ------------------------ |
| `model`       | string         | 是  | 固定为 `claude-opus-4-5`    |
| `messages`    | array          | 是  | `{ role, content }` 对象数组 |
| `max_tokens`  | integer        | 否  | 最大生成 Token 数             |
| `temperature` | float          | 否  | `0`–`2`，控制随机性，默认 `1`     |
| `stream`      | boolean        | 否  | 开启 SSE 流式传输，默认 `false`   |
| `top_p`       | float          | 否  | 核采样阈值，默认 `1`             |
| `stop`        | string / array | 否  | 触发停止生成的序列                |
