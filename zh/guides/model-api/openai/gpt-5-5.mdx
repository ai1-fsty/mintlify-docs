GPT-5.5 通过 Starrise AI 以 OpenAI Responses API (`/v1/responses`) 提供服务。这是 OpenAI 最强的前沿模型，专为最复杂的专业工作和编程任务设计。

## 核心能力

* **Responses API** — 使用新版 `/v1/responses` 端点，用 `input` 替代 `messages`
* **高级推理** — 支持推理工作量级别：none、low、medium（默认）、high、xhigh
* **详细度控制** — 设置响应详细程度为 low、medium 或 high
* **流式输出** — 通过 SSE 实现实时 Token 流式传输
* **工具调用** — 支持函数调用、联网搜索、文件检索和计算机操作

## 快速示例

<CodeGroup>
  ```bash cURL
  curl https://ai.starrise.tech/v1/responses \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{
      "model": "gpt-5.5",
      "input": [
        { "role": "user", "content": "用简单的语言解释量子纠缠。" }
      ],
      "reasoning": {
        "effort": "high",
        "summary": "auto"
      },
      "text": {
        "format": { "type": "text" },
        "verbosity": "medium"
      },
      "store": true
    }'
  ```

  ```python Python
  from openai import OpenAI

  client = OpenAI(
      api_key="YOUR_API_KEY",
      base_url="https://ai.starrise.tech/v1"
  )

  response = client.responses.create(
      model="gpt-5.5",
      input=[
          {"role": "user", "content": "用简单的语言解释量子纠缠。"}
      ],
      reasoning={"effort": "high", "summary": "auto"},
      text={"format": {"type": "text"}, "verbosity": "medium"},
      store=True
  )

  print(response.output[0].content[0].text)
  ```

  ```python 流式输出
  from openai import OpenAI

  client = OpenAI(
      api_key="YOUR_API_KEY",
      base_url="https://ai.starrise.tech/v1"
  )

  stream = client.responses.create(
      model="gpt-5.5",
      input=[
          {"role": "user", "content": "写一首关于大海的短诗。"}
      ],
      stream=True
  )

  for event in stream:
      if hasattr(event, 'delta'):
          print(event.delta, end="")
  ```
</CodeGroup>

## 参数说明

| 参数                  | 类型             | 必填 | 说明                                                                   |
| ------------------- | -------------- | -- | -------------------------------------------------------------------- |
| `model`             | string         | 是  | 固定为 `gpt-5.5`                                                        |
| `input`             | array          | 是  | `{ role, content }` 对象数组                                             |
| `stream`            | boolean        | 否  | 开启 SSE 流式传输，默认 `false`                                               |
| `temperature`       | float          | 否  | `0`–2，控制随机性，默认 `1`                                                   |
| `max_output_tokens` | integer        | 否  | 最大输出 Token 数，最小值 `16`                                                |
| `stop`              | string / array | 否  | 触发停止生成的序列                                                            |
| `reasoning`         | object         | 否  | `{ effort, summary }` — effort 可选：none / low / medium / high / xhigh |
| `text`              | object         | 否  | `{ format, verbosity }` — 控制输出格式和详细度                                 |
| `tools`             | array          | 否  | 模型可调用的工具列表                                                           |
| `store`             | boolean        | 否  | 存储响应以供后续检索，默认 `true`                                                 |
