HappyHorse 1.0 T2V 是阿里 HappyHorse 视频生成系列的文生视频模型。使用**异步任务接口**——提交任务后轮询直至完成。

## 核心能力

* **文生视频** — 仅靠文本描述生成视频
* **分辨率** — 720P / 1080P
* **宽高比** — 16:9、9:16、1:1、4:3、3:4
* **时长** — 3-15 秒
* **水印控制** — 可选关闭右下角 "Happy Horse" 水印

## 工作流

```
1. POST /v1/video/generations  →  task_id
2. GET  /v1/video/generations/{task_id}  →  轮询直至 SUCCESS / FAILURE
3. 读取 data.result_url  （有效期 24 小时）
```

## 快速示例

<CodeGroup>
  ```bash cURL
  # 第一步 — 创建任务
  curl https://ai.starrise.tech/v1/video/generations \
    -H 'Authorization: Bearer YOUR_API_KEY' \
    -H 'Content-Type: application/json' \
    -d '{
      "model": "happyhorse-1.0-t2v",
      "prompt": "一只猫在草地上奔跑",
      "parameters": {
        "resolution": "720P",
        "ratio": "16:9",
        "duration": 5,
        "watermark": false
      }
    }'

  # 第二步 — 轮询直至完成
  curl https://ai.starrise.tech/v1/video/generations/asyntask_xxx \
    -H 'Authorization: Bearer YOUR_API_KEY'
  ```

  ```python Python
  import requests, time

  headers = {"Authorization": "Bearer YOUR_API_KEY"}

  resp = requests.post(
      "https://ai.starrise.tech/v1/video/generations",
      headers=headers,
      json={
          "model": "happyhorse-1.0-t2v",
          "prompt": "一只猫在草地上奔跑",
          "parameters": {
              "resolution": "720P",
              "ratio": "16:9",
              "duration": 5,
              "watermark": False,
          }
      }
  )
  task_id = resp.json()["task_id"]

  while True:
      r = requests.get(
          f"https://ai.starrise.tech/v1/video/generations/{task_id}",
          headers=headers,
      ).json()
      status = r["data"]["status"]
      if status == "SUCCESS":
          print("视频 URL:", r["data"]["result_url"])
          break
      if status == "FAILURE":
          print("失败:", r["data"]["fail_reason"])
          break
      time.sleep(15)
  ```
</CodeGroup>

## 参数说明

可选参数可放在请求顶层，也可放在 `parameters` 对象中——两种写法等价。

| 参数           | 类型      | 必填 | 说明                                        |
| ------------ | ------- | -- | ----------------------------------------- |
| `model`      | string  | 是  | 必须为 `happyhorse-1.0-t2v`                  |
| `prompt`     | string  | 是  | 视频描述文本                                    |
| `resolution` | string  | 否  | `720P` / `1080P`，默认 `1080P`               |
| `ratio`      | string  | 否  | `16:9`、`9:16`、`1:1`、`4:3`、`3:4`，默认 `16:9` |
| `duration`   | integer | 否  | 视频时长 3-15 秒，默认 `5`                        |
| `watermark`  | boolean | 否  | 默认 `true`，设为 `false` 关闭水印                 |
| `seed`       | integer | 否  | 随机种子 `[0, 2147483647]`                    |

## 计费

按 **分辨率档位 × 用户请求时长** 计费，预扣金额即最终金额：

* `duration: 5` → 收 5 秒的钱（即使上游实际生成 6 秒也按 5 秒）
* 失败任务自动退费
