---
title: "Kling 查询自定义主体"
description: "通过 task_id 查询自定义主体创建任务的状态和结果。"
---

创建自定义主体后，使用此接口轮询任务状态，直到状态变为 `succeed` 或 `failed`。成功后，响应中包含主体详情，包括 `element_id`。

## 快速示例

<CodeGroup>
  ```bash cURL
  curl https://ai.starrise.tech/kling/v1/general/advanced-custom-elements/863121016086724692 \
    -H "Authorization: Bearer YOUR_API_KEY"
  ```

  ```python Python
  import requests
  import time

  task_id = "863121016086724692"

  while True:
      response = requests.get(
          f"https://ai.starrise.tech/kling/v1/general/advanced-custom-elements/{task_id}",
          headers={"Authorization": "Bearer YOUR_API_KEY"}
      )
      result = response.json()
      status = result["data"]["task_status"]
      print(f"状态: {status}")

      if status == "succeed":
          elements = result["data"]["task_result"]["elements"]
          print(f"Element ID: {elements[0]['element_id']}")
          break
      elif status == "failed":
          print("主体创建失败")
          break

      time.sleep(3)
  ```
</CodeGroup>

## 返回字段

| 字段                               | 说明                                 |
| -------------------------------- | ---------------------------------- |
| `data.task_id`                   | 任务 ID                              |
| `data.task_status`               | `submitted` / `succeed` / `failed` |
| `data.task_result.elements[]`    | 创建的主体列表                            |
| `elements[].element_id`          | 主体 ID（用于删除操作）                      |
| `elements[].element_name`        | 主体名称                               |
| `elements[].element_description` | 主体描述                               |
| `elements[].element_type`        | 固定 `image_refer`                   |
| `elements[].element_image_list`  | 上传图片的 CDN URL                      |
| `elements[].status`              | `succeed` / `failed`               |
| `elements[].tag_list`            | 标签列表，包含 id 和 name                  |
| `data.created_at`                | 创建时间（毫秒时间戳）                        |
| `data.updated_at`                | 更新时间（毫秒时间戳）                        |
