---
title: "Kling 查询官方预设主体列表"
description: "查询 Kling 官方提供的预设主体，所有用户均可直接使用。"
---

Kling 提供了一套官方预设主体（场景、人物等），无需自己创建即可在视频生成中使用。这些主体的 `owned_by` 固定为 `"kling"`。

## 快速示例

<CodeGroup>
  ```bash cURL
  curl "https://ai.alad.com/kling/v1/general/advanced-presets-elements?pageNum=1&pageSize=10" \
    -H "Authorization: Bearer YOUR_API_KEY"
  ```

  ```python Python
  import requests

  response = requests.get(
      "https://ai.alad.com/kling/v1/general/advanced-presets-elements",
      headers={"Authorization": "Bearer YOUR_API_KEY"},
      params={"pageNum": 1, "pageSize": 10}
  )

  data = response.json()
  for item in data["data"]:
      for el in item.get("task_result", {}).get("elements", []):
          print(f"{el['element_id']}: {el['element_name']} — {el['tag_list'][0]['name']}")
  ```
</CodeGroup>

## 请求参数

| 参数         | 类型      | 说明        |
| ---------- | ------- | --------- |
| `pageNum`  | integer | 页码，从 1 开始 |
| `pageSize` | integer | 每页数量      |

## 与自定义主体的区别

| 字段         | 自定义主体   | 预设主体      |
| ---------- | ------- | --------- |
| `owned_by` | 您的用户 ID | `"kling"` |
| 可删除        | 是       | 否         |
| 可编辑        | 是       | 否         |
