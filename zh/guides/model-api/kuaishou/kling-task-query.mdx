Kling 任务查询接口用于检查任何可灵生成任务的状态并获取结果。这是所有可灵模型（Omni、文生视频、图生视频等）共用的查询端点。

## 要点

* **通用查询** — 适用于所有可灵任务类型：视频、图片、口型同步、虚拟试穿
* **异步工作流** — 先创建任务，然后轮询此接口直到完成
* **Action 参数** — 必须与原始创建请求的任务类型匹配
* **进度追踪** — 返回进度百分比和状态更新

## Action 类型

| Action                  | 说明              |
| ----------------------- | --------------- |
| `text2video`            | 文生视频和 Omni 视频任务 |
| `image2video`           | 图生视频任务          |
| `generations`           | 图片生成任务          |
| `lip-sync`              | 口型同步视频任务        |
| `kolors-virtual-try-on` | 虚拟试穿图片任务        |

## 快速示例

<CodeGroup>
  ```bash cURL
  curl https://ai.alad.com/kling/v1/videos/text2video/860260753860210752 \
    -H "Authorization: Bearer YOUR_API_KEY"
  ```

  ```python Python
  import requests
  import time

  task_id = "860260753860210752"
  action = "text2video"

  while True:
      response = requests.get(
          f"https://ai.alad.com/kling/v1/videos/{action}/{task_id}",
          headers={"Authorization": "Bearer YOUR_API_KEY"}
      )

      result = response.json()
      status = result["data"]["status"]
      progress = result["data"]["progress"]
      print(f"状态: {status}, 进度: {progress}")

      if status in ["SUCCESS", "FAILED"]:
          break

      time.sleep(5)

  if status == "SUCCESS":
      videos = result["data"]["data"]["data"]["task_result"]["videos"]
      for video in videos:
          print(f"视频 URL: {video['url']}")
  ```
</CodeGroup>

## 响应字段

| 字段                           | 类型      | 说明                                            |
| ---------------------------- | ------- | --------------------------------------------- |
| `code`                       | string  | 响应码（`success`）                                |
| `data.task_id`               | string  | 任务 ID                                         |
| `data.status`                | string  | `SUBMITTED`、`PROCESSING`、`SUCCESS` 或 `FAILED` |
| `data.progress`              | string  | 进度百分比（如 `10%`、`100%`）                         |
| `data.submit_time`           | integer | 提交 Unix 时间戳                                   |
| `data.start_time`            | integer | 开始处理 Unix 时间戳                                 |
| `data.finish_time`           | integer | 完成 Unix 时间戳                                   |
| `data.data.data.task_result` | object  | 包含 `videos` 或 `images` 数组及下载 URL              |
