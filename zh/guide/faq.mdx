## 账户与计费

**如何创建账户？**
在 [{{api-origin}}]({{api-origin}}) 使用邮箱或 OAuth（GitHub、Discord 等）注册。

**如何计费？**
对话模型按 Token 计费（输入 + 输出）。图片、视频和音频模型按生成次数计费。各模型定价见控制台 **模型定价** 页面。

**可以设置消费上限吗？**
可以。在 API 令牌上设置配额限制来控制消费。配额耗尽后令牌将停止工作。

**支持哪些支付方式？**
请在控制台 **充值** 页面查看可用支付方式。根据地区不同，我们支持多种支付选项。

**未使用的余额会过期吗？**
账户余额不会过期。令牌特定的配额可能设有过期时间。

## API 与使用

**Base URL 是什么？**
您的 API 地址显示在控制台首页，通常格式为 `{{api-base-url}}`。

**有哪些可用模型？**
在控制台 **模型定价** 页面查看完整列表。也可调用 `GET /v1/models` 查看当前令牌可用的模型。

**如何从 OpenAI 切换过来？**
将 `base_url` 改为 `{{api-base-url}}`，`api_key` 改为 Starrise AI 令牌。使用 OpenAI SDK 无需修改代码。

**支持流式输出吗？**
支持。在对话补全请求中设置 `"stream": true` 即可接收 Server-Sent Events（SSE）。

**如果供应商服务中断怎么办？**
错误响应会包含明确的消息指示问题。可在控制台配置备用渠道以提高可用性。

**速率限制如何工作？**
速率限制因模型和账户等级而异。超出限制时 API 返回 `429 Too Many Requests`。建议实现指数退避策略。

## 安全

**如何安全存储 API 密钥？**
使用环境变量。切勿在源代码中硬编码 API 密钥或暴露在前端代码中。

**可以限制 API 密钥使用吗？**
可以。每个 API 令牌可设置 IP 白名单、模型限制和配额上限。

**API 密钥泄露了怎么办？**
立即在控制台删除被泄露的令牌并创建新令牌。

**数据传输是否加密？**
是。所有 API 流量使用 HTTPS/TLS 加密。

## 技术

**API 兼容 OpenAI 吗？**
完全兼容。对话、图片、音频、向量和模型端点均遵循 OpenAI API 规范。

**可以使用 Anthropic SDK 吗？**
可以。设置 `ANTHROPIC_BASE_URL` 为 `{{api-origin}}`，使用 Starrise 令牌作为 API 密钥。

**可以使用 Google Gemini SDK 吗？**
可以。配置客户端使用 `{{api-origin}}` 作为 API 端点，使用 Starrise 令牌认证。

**如何获取帮助？**
通过控制台联系支持，或查看 [快速开始](/zh/guide/quick-start) 指南入门。
